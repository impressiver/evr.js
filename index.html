Getting Started

  1. Download EVR.js (TAR or Github) and put the files somewhere within your project resource path.
  2. Include the CSS tag somewhere in the HEAD of the html: 
    <head>
  ...
  <link href="/path/to/css/evr.css" rel="stylesheet">
  ...
</head>
  3. Include the JS for jQuery and EVR at the end of the BODY: 
    <body>
  ...
  <script src="/path/to/js/jquery.js"></script>
  <script src="/path/to/js/jquery.evr.js"></script>
</body>
    Heads up! The same jquery.evr.js file will also work with browser-based AMD script loaders, like RequireJS.
  4. Instantiate EVR: 
    $(function() {
  // Load EVR!
  $(window).evr('start');
});
    Heads up! Make sure to place this code somewhere after the script tags you added in step 3.
  5. And when you're done, make it stop: 
    $(window).evr('stop');

Options

The default settings are good enough to get you started, but there are plenty of settings to explore. Keep scrolling to find the full list of options.

Setting global options:

These overrides will apply to all instances of EVR.

  1. During initialization: 
    // Options to override
var options = {
  ...
}

// Initialize the plugin, the options will be merged with the defaults and apply to any guide loaded
$(window).evr('start', options);
  2. After initialization: 
    // These options will be merged with the existing options for an instance of EVR
$(window).evr('options', {
  ...
});

    @return {jQuery} jQuery selector
